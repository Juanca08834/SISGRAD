
# Documentación de arquitectura

Este documento describe la solución con microservicios y orientación a eventos para el
**Sistema de Gestión de Trabajos de Grado (FIET)**. Se incluyen diagramas C4, patrones de diseño,
y trazabilidad con las historias de usuario del segundo corte.

## Diagramas (C4 + UML)

- `c4-container.puml`: Diagrama de contenedores (C4) con gateway, servicios y dependencias.
- `c4-context.puml`: Diagrama de contexto con actores (Docente, Coordinador, Jefatura, Estudiante).
- `uml-component.puml`: Componentes principales por microservicio.

Renderice con PlantUML (IntelliJ plugin) o en https://www.plantuml.com/plantuml/ .
